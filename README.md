# Raid Damage Tracker

A RuneLite plugin that tracks your personal damage dealt to each boss across all three OSRS raids, with live DPS calculations and a clean side panel UI.

## Features

- **Chambers of Xeric (CoX)** — tracks all bosses including Tekton, Vasa, Vespula, Muttadile, Skeletal Mystics, Shamans, Ice Demon, Guardians, and Olm
- **Theatre of Blood (ToB)** — tracks Maiden, Bloat, Nylocas, Sotetseg, Xarpus, and Verzik
- **Tombs of Amascut (ToA)** — tracks Apmeken, Baba, Kephri, Zebak, Akkha, Het, and Wardens
- Live damage counter per boss, updating in real time as you deal damage
- NPC card appears as soon as the boss room is entered
- Boss marked as dead when it despawns

## DPS Modes

### Boss Split
Shows a DPS log at the bottom of the panel with each completed boss's damage and DPS, calculated from your first hit on that boss to its death.

### Raid Overall
Shows a running overall DPS across the entire raid, with the timer paused during inter-room transitions so only active combat time counts.

## Usage

1. Enter a raid — the panel will show the current raid type and mode
2. Damage to each boss is tracked automatically
3. Use the **Reset** button to clear all data mid-raid if needed

## Configuration

Access the config panel via the RuneLite settings wrench:

- **DPS Mode** — choose between `Boss Split` and `Raid Overall`
